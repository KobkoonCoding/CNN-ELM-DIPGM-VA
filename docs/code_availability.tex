\section*{Code Availability}
The source code that implements DIPGM-VA and reproduces all experiments in
this paper is openly available on GitHub at
\url{https://github.com/KobkoonCoding/CNN-ELM-DIPGM-VA} and permanently
archived on Zenodo at \url{https://doi.org/10.5281/zenodo.XXXXXXX}
(DOI minted from the corresponding GitHub release tag). The repository
contains a single end-to-end Jupyter notebook, a pinned
\texttt{requirements.txt}, a Conda \texttt{environment.yml}, and an MIT
license.

\section*{Data Availability}
The Kermany \emph{et al.} (2018) chest X-ray pneumonia dataset used in this
study is publicly available on Kaggle at
\url{https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia}.
The notebook downloads the dataset automatically via the
\texttt{kagglehub} API; no additional preprocessing is required.
